Administration

Commands that only work for server administrators.

Command Description Usage
Setwelcomemessage Set the server's welcome message. p!setwelcomemessage <message>
Disablewelcomemessage Disable the server's welcome message. p!disablewelcomemessage
Setmodlog Set the channel for all moderator actions to be logged in. p!setmodlog <#channel>
SetPrefix Set the command prefix for the server. p!setprefix <prefix>
Setmutedrole Set the muted role for the server. p!setmutedrole <@role>

Botowners

Commands that only work for the creator(s) of Pixel.

Command Description Usage
Eval Evaluate JavaScript code. p!eval <code>
Exec Execute batch/bash script. p!exec <code>
Setgame Sets the bot's playing game status. p!setgame <game>
Kill Terminates the Node process (turns the bot off). p!kill  NOTE: Pixel will ask for confirmation when using this command.

General (Member group)

General random commands for members.

Command Description Usage
Groups View the command groups in Pixel. p!groups
Createpoll Create a poll. p!createpoll <nameOfPoll> <choice1~choice2> <daysToLast> <true/false> <true/false>
Embed Embed the provided text. p!embed <text>
48yearman Say the famous 48 year man from somalia copypasta. p!48yearman
Globalxp View your/anyone else's global XP. p!globalxp [@member]
Globalleaderboard View the highest levelled users globally. p!globallb
Help View all commands. p!help [rule]
Info View your/anyone else's XP. p!xp [@member]
Invite View the Support Server invite link, the GitHub link and the bot invite link. p!invite
Leaderboard View the highest levelled members. p!lb
Modroles View the server's moderator roles. p!modroles
Ping View the message and heartbeat latency. p!ping
Poll View the info of a specific poll. p!poll <pollNumber>
Removepoll Remove a poll (if you're the creator of it.) p!removepoll <pollNumber>
Reply Check if the bot is online. p!reply
Skills View your/anyone else's skills. p!skills [@member]
Statistics View Pixel's statistics (uptime, etc.) p!statistics
Upgradeskill Upgrade a/multiple skills. p!upgrade <skill(s)>
Vote Vote on a poll. p!vote <poll> <choice>

Owners

These commands only work for server owners.

Command Description Usage
Addmodrole Add a moderator role. p!addmodrole <role> <permissionLevel>
Removemodrole Remove a moderator role. p!removemodrole <role>
Modifymodrole Edit a moderator role. p!modifymodrole <role> <permissionLevel>

Moderators

These commands only work for moderators (members with a permission level of 1 or higher)

Command Description Usage
Ban Ban a member. p!ban <@member> [reason]
Clear Delete 𝓍 amount of messages. p!clear <numberOfMessages> [reason]
Kick Kick a member. p!kick <@member> [reason]
Mute Mute a member. p!mute <@member> <numberOfHours> [reason]
Quickpoll Make a quick poll (yes/no poll with reactions.) p!qp <pollName>
Softban Bans a member to delete their messages, then unbans them. p!softban <@member> <numberOfDays> [reason]
Unban Unban a member. p!unban <@member> [reason]
Unmute Unmute a member. p!unmute <@member> [reason]
